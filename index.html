Atributos del Modelo de Cajas

El modelo de cajas de CSS está compuesto por:

  • margin: Espacio exterior alrededor del elemento
  • border: Borde que rodea el padding y contenido
  • padding: Espacio interior entre el borde y el contenido
  • content: El contenido real del elemento

Propiedades Principales de Flexbox

Las propiedades más importantes de Flexbox son:

  • justify-content: Alinea elementos en el eje principal (center, space-between, space-around)
  • align-items: Alinea elementos en el eje transversal (center, flex-start, flex-end)
  • flex-wrap: Controla si los elementos pueden saltar a una nueva línea (wrap, nowrap)
